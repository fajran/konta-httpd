Menjalankan server

Untuk menjalakankan program server, cukup panggil program server yang terbentuk dari hasil kompilasi.

Contoh: 

$ ./konta-httpd
Konta-HTTP Server running on port 8000...

Setelah server berjalan, Anda bisa mencoba mengakses server melalui browser kesayangan Anda.

Secara default, direktori dokumen html ada pada direktori htdocs/. Anda dapat meletakkan beberapa file yang dapat Anda gunakan untuk mencoba server kami ini.
